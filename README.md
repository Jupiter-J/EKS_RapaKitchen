Web Application Hosts on EKS Workshop

###  구현 환경
- kubernetes : v 1.23.9
- eksctl : 0.115.0
- Region : ap-northeast-2 (서울)   
- AZ
  - ap-northeast-2a
  - ap-northeast-2b
  - ap-northeast-2c
  - ap-northeast-2d 
  
- **VPC CIDR 대역**
  - 192.168.0.0/16
- **Worker Public Subnet CIDR 대역 - 각 가용영역에 하나씩 배치**
  - 192.168.252.0/24
  - 192.168.253.0/24
  - 192.168.254.0/24
  - 192.168.255.0/24
- **Worker Private Subnet CIDR 대역  - 각 가용영역에 하나씩 배치**
  - 192.168.0.0/20
  - 192.168.16.0/20
  - 192.168.32.0/20
  - 192.168.48.0/20
- **Database Private Subnet CIDR 대역 - a, b 가용영역에 하나씩 배치**
  - 192.168.250.0/24
  - 192.168.251.0/24
- **Bastion Host : 가용영역 A Public Subnet에 배치**
  - 프로젝트 작업 : cloud9 사용
